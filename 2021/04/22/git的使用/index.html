git的使用

1.克隆github上的项目：

1.创建一个文件夹
2.在这个文件夹打开git客户端
3.输入 git init,生成一个.git文件，你就可以文件靠这个管理这个文件夹

4.输入git clone +仓库地址

成功：

2.上传项目或者更改到github：

1)、git init 初始化仓库，将这个目录变成git可以管理的仓库
2)、 然后使用git add . 将文件添加到暂存区里面去，注意后边的 .
3)、然后执行git commit -m ‘描述内容’ 命令将文件提交到仓库
4)、然后关联git仓库，因为此时git还不知道要将我们的项目添加到哪里去。接下来执行命令：git remote add origin 远程仓库的地址
5)、然后执行git pull –rebase origin master命令，将文件与远程仓库进行合并
6)、然后将本地文件推送到gitee仓库中，使用命令：git push -u origin master 命令
